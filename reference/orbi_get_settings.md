# Get all isoorbi package settings

**\[deprecated\]**

`orbi_get_settings()` was renamed
[`orbi_get_options()`](https://isoorbi.isoverse.org/reference/orbi_options.md)
as part of `isoorbi` switching from 'settings' to 'options' to be
consistent with base R naming conventions

## Usage

``` r
orbi_get_settings(pattern = NULL)
```

## Arguments

- pattern:

  passed on to
  [`orbi_get_options()`](https://isoorbi.isoverse.org/reference/orbi_options.md)
