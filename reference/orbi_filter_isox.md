# Filter isox files

**\[deprecated\]** `orbi_filter_isox()` was renamed
[`orbi_filter_files()`](https://isoorbi.isoverse.org/reference/orbi_filter_files.md)
to incorporate its wider scope for filtering both isox and raw files
data.

## Usage

``` r
orbi_filter_isox(...)
```

## Arguments

- ...:

  arguments passed on to
  [`orbi_filter_files()`](https://isoorbi.isoverse.org/reference/orbi_filter_files.md)
