# Set package settings

**\[deprecated\]**

`orbi_set_settings()` was renamed
[`orbi_options()`](https://isoorbi.isoverse.org/reference/orbi_options.md)
as part of `isoorbi` switching from 'settings' to 'options' to be
consistent with base R naming conventions

## Usage

``` r
orbi_set_settings(...)
```

## Arguments

- ...:

  named arguments to set specific options, passed on to
  [`orbi_options()`](https://isoorbi.isoverse.org/reference/orbi_options.md)
